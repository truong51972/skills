# Celery Worker Skill

Production Celery 5 worker architecture guidance for task design, queue routing,
broker-aware reliability, lifecycle management, and testing.

Use this skill when you need to build or review:

- Celery app setup, broker settings, queues, and routes
- thin task wrappers, retries, acknowledgements, and idempotency
- worker process lifecycle and dependency-injection resources
- Django + Celery integration
- realistic unit, integration, and worker-backed tests
