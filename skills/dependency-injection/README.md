# Dependency Injection Skill

Modern Python dependency-injection guidance for FastAPI, Celery, CLI, tests, and
shared application composition.

Use this skill when you need to design or refactor:

- `dependency-injector` containers, providers, and resource lifecycle
- Pydantic v2 settings injection
- FastAPI `Depends`, lifespan, and app-state integration
- Celery worker DI bootstrap and fork-safe resources
- test overrides, fakes, and fixture cleanup
